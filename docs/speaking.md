# Speaking

Đây là nội dung mẫu cho kỹ năng Speaking. Thay nội dung file này (docs/speaking.md)
bằng chủ đề, dàn ý trả lời mẫu, hoặc ghi chú ôn luyện Speaking của bạn.

Ví dụ bạn có thể dán:
- Chủ đề thường gặp trong Speaking VSTEP
- Dàn ý/khung câu trả lời mẫu
- Từ vựng, cách diễn đạt hay dùng khi nói
