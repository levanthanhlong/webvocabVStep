# Writing

Đây là nội dung mẫu cho kỹ năng Writing. Thay nội dung file này (docs/writing.md)
bằng đề bài, dàn ý mẫu, bài viết mẫu, hoặc ghi chú ôn luyện Writing của bạn.

Ví dụ bạn có thể dán:
- Đề bài Writing Task 1 / Task 2 của VSTEP
- Dàn ý và cấu trúc bài mẫu
- Từ nối, cụm từ hay dùng trong Writing
